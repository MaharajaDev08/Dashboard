# 📊 E-Commerce Sales Dashboard

An interactive dashboard built with **Python (Plotly)** to visualize key e-commerce metrics — sales, profit, returns, and revenue trends — to support data-driven decision-making.

## 🔗 Live Dashboard
👉 [View Live Dashboard](https://maharajadev08.github.io/ecommerce-dashboard/ecommerce_dashboard.html)

## 📁 Project Files
- `dashboard.py` — Python script that generates the interactive dashboard using Plotly
- `ecommerce_dashboard.html` — Rendered output (the live dashboard)

## 📈 Features
- Customer distribution chart
- Sales by category (pie chart)
- Profit by category
- Monthly returns tracking
- Revenue vs Orders comparison

## 🛠️ Tech Stack
- Python
- Plotly (for interactive charts)
- Pandas (for data handling)

## 🚀 How to Run Locally
```bash
pip install -r requirements.txt
python dashboard.py
```
This will generate `ecommerce_dashboard.html`, which you can open in any browser.
